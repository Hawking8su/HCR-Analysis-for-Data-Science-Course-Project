Variables Explanation
=================

Dataset 1: HCRData
1 "activityLabel": descriptive activity label name for each obervation          
2 "subjectIndex": subject index for each observation          
3..end: the rest of the variables are the mean() and std() measurements from the
  available features in the dataset directory.

=================

Dataset 2: Avg_HCRData_for_Act_Sub
1 "activityLabel": descriptive activity label name for each obervation          
2 "subjectIndex": subject index for each observation  
3...end: the average of each variable for each activity and each subject in the HCRData.
